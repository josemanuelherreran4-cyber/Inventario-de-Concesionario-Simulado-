# Inventario-de-Concesionario-Simulado-
Codigo en R de los movimientos de un concesionario en expansión
Este dataset simula la operación integral de un concesionario de vehículos moderno ("Gestion Consecionario"), abarcando desde la adquisición del inventario hasta la venta final. Ha sido diseñado para reflejar desafíos del mundo real, como la gestión de stock de vehículos nuevos y seminuevos, la aplicación de descuentos comerciales y la volatilidad de la demanda según el tipo de combustible (Gasolina, Híbrido, Eléctrico).

#Contenido y Estructura


El dataset sigue una arquitectura de Esquema en Estrella (Star Schema), optimizada para herramientas de analítica como Power BI, R y Python.
